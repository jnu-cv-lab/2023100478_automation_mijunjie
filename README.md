OpenCV 基础图像处理入门
一个适合初学者的 OpenCV 基础图像处理教程，包含**图像读取、信息查看、显示、灰度转换、保存、裁剪**等 6 个核心任务，代码简洁易懂，可直接运行学习。

## 📋 实现功能（6 大任务）
1. **使用 OpenCV 读取本地图片**
2. **打印图像基础信息**：宽度、高度、通道数、数据类型
3. **显示原始图像**（支持 OpenCV / Matplotlib 两种方式）
4. **彩色图像转灰度图像**并展示效果
5. **保存处理后的灰度图**到本地
6. **NumPy 基础操作**：读取像素值 + 裁剪图像左上角区域并保存

## 🧩 环境依赖
需要安装以下 Python 库：
```bash
pip install opencv-python numpy matplotlib
```

## 🚀 使用方法
1. 将你的测试图片放入项目根目录，命名为 `test.jpg`（或修改代码中的图片路径）
2. 直接运行 Python 脚本
3. 查看终端输出的图像信息，以及自动保存的处理结果图片

## 📂 项目文件结构
```
opencv-basic-image-processing/
├── main.py          # 主程序代码（完整任务实现）
├── test.jpg         # 你的测试图片
├── gray_result.jpg  # 程序自动保存：灰度图
├── cropped_result.jpg # 程序自动保存：裁剪后的图像
└── README.md        # 项目说明文档
```

## 💡 核心知识点
- OpenCV 读取图像默认是 **BGR 色彩空间**，Matplotlib 显示需要转换为 RGB
- 图像在 Python 中以 **NumPy 数组** 形式存储，可通过索引操作像素/区域
- 图像坐标格式：`图像[行(高度y), 列(宽度x)]`
