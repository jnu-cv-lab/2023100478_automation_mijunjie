# 计算机视觉实验三：图像下采样、插值恢复与频域分析
**课程**：计算机视觉 | 数字图像处理  
**内容**：下采样 + 三种插值恢复 + FFT 傅里叶分析 + DCT 变换分析

---

## 📌 实验任务
1. 读取灰度图像（Lena 标准图）
2. **下采样**：
   - 直接下采样（1/2）
   - 高斯预滤波后下采样（抗混叠）
3. **图像恢复**（三种插值方法）：
   - 最近邻插值
   - 双线性插值
   - 双三次插值
4. **空间域评价**：
   - 显示原图、缩小图、恢复图
   - 计算 MSE、PSNR
5. **频域分析**：
   - 傅里叶变换 FFT（中心化 + 对数幅度谱）
   - 离散余弦变换 DCT（系数图 + 低频能量占比）
6. 比较不同插值方法的图像质量与频域差异

## 📊 实验结果
1. **图像恢复质量**
   - 质量排序：**双三次 > 双线性 > 最近邻**
   - 最近邻：块效应、锯齿明显
   - 双线性：图像平滑，轻微模糊
   - 双三次：边缘保留最好，质量最优

2. **傅里叶变换（FFT）**
   - 原图：高频信息丰富
   - 下采样后：高频成分丢失
   - 恢复图：无法还原真实高频，频谱外围变暗

3. **DCT 变换**
   - 原图低频能量占比极高
   - 恢复图能量更集中于低频
   - 高频系数显著衰减，说明细节丢失

## 📝 实验结论
- 下采样前进行**高斯滤波**可有效减少混叠失真。
- **双三次插值**在图像恢复中表现最优。
- 下采样会丢失高频信息，插值仅能平滑填充，无法还原真实细节。
- FFT 与 DCT 能直观反映图像频域变化，是评价图像质量的重要工具。
